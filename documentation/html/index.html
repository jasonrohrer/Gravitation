Note: To build from source, you must have the SDL development library installed.

Control notes:
  • Press any key to start the game.
  • Use arrow keys to move character (left and right).
  • Use space bar to jump.
  • F to toggle fullscreen mode.
  • B to adjust screen blow-up factor.
  • Q or ESC to quit.
Display notes:
  • Default full-screen resolution is 640x480.
  • Some monitors (like LCD?) don't do well at low resolutions (blurry pixels).
  • Edit the files in the settings folder to change the full screen resolution (try 1024 for screenWidth.ini and 768 for screenHeight.ini).
  • Put a zero in fullscreen.ini to make the game start in windowed mode.
